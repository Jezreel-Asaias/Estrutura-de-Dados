# 🐍 Estruturas de Dados com Python
## Descrição do projeto
Repositório dedicado ao estudo e implementação de algoritmos e estruturas de dados, utilizando Python 3 e as ferramentas de produtividade do VS Code.
O foco aqui é entender a eficiência por trás das abstrações, analisando a complexidade de tempo e espaço das operações.

## 🛠️ Ambiente de Desenvolvimento
1- Linguagem: Python 3

2- Editor: Visual Studio Code (VS Code)

3- Extensões recomendadas: * Python (Microsoft): Para linting e depuração.

4- Python Preview: Para visualizar a execução do código passo a passo.

## 🚀 Como Rodar os Exemplos
1- Certifique-se de ter o Python instalado:
```bash
python --version
```

2- Clone este repositório:
```bash
git clone https://github.com/Jezreel-Asaias/Estrutura-de-Dados.git
```

3- Execute um arquivo específico:
No VS Code, abra o arquivo .py e pressione F5 ou execute via terminal:
```bash
python nome_do_arquivo.py
```
